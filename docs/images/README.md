# Screenshots (optional)

Add PNG or WebP files here to show in the main [README](../README.md).

Suggested captures:

| File | What to show |
|------|----------------|
| `menu-bar.png` | Menu bar item idle and/or spinner while checking |
| `results-overlay.png` | Results card over a dimmed desktop |
| `settings.png` | Settings window (LM Studio + shortcut) |

After exporting, link them from the README **Screenshots** section, for example:

Keep captures free of personal data (model names in screenshots are fine if you’re comfortable).
